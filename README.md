# 💻 Help Informática

Website institucional desenvolvido para apresentar os serviços da Help Informática, com foco em suporte técnico, manutenção de computadores e soluções em tecnologia.

---

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de oferecer uma presença digital profissional para a Help Informática, facilitando o acesso dos clientes aos serviços oferecidos e fortalecendo a credibilidade da marca.

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 🎯 Funcionalidades

- Apresentação dos serviços  
- Layout responsivo  
- Seção de contato  
- Navegação simples e intuitiva
